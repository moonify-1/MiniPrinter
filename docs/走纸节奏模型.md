# MiniPrinter 走纸节奏模型

更新时间：2026-04-26 21:34:32

## 1. 规格推导

- 机芯打印密度：`8 dots/mm`。
- 因此一行点距：`1 / 8 = 0.125 mm`。
- 机芯步进距离：`0.0625 mm/step`。
- 所以每打印一行后应走：`0.125 / 0.0625 = 2 steps`。

## 2. 当前固件参数

| 参数 | 默认值 | 说明 |
|---|---:|---|
| `stepsPerPrintLine` | `2` | 每打印一行后走 2 个整步。 |
| `startPhaseIntervalUs` | `5000` | 起步阶段相位切换间隔，保守偏慢。 |
| `runPhaseIntervalUs` | `3000` | 当前打印走纸默认相位切换间隔。 |
| `fastPhaseIntervalUs` | `2000` | 预留较快节奏，不作为真实高速打印验收依据。 |

## 3. 安全边界

- 真实走纸方向仍需用户通过 `POST /api/v1/factory/motor-test` 低速小步数确认。
- 未确认电机方向前，不应进入真实连续打印。
- 如果发现走纸方向相反，应修正 DRV8833 相序或方向配置，再继续验证。
