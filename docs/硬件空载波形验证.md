# MiniPrinter 空载波形验证记录

更新时间：2026-04-26 21:34:32

## 1. 验证边界

- 本文只准备空载验证步骤和记录模板。
- Codex 不能替代示波器、逻辑分析仪或万用表实测。
- 在完成本页所有安全态和空载波形确认前，不得接入真实加热打印。

## 2. 前置条件

| 项目 | 要求 | 用户实测 |
|---|---|---|
| 固件宏 | 可使用 `MP_ENABLE_WIFI=1`；需要测打印头 GPIO 时再临时加 `MP_ENABLE_HW_THERMAL_HEAD=1` | 待填写 |
| VH | `G_VH` 已确认高通低断，但 safe-off 后必须实测为 LOW | 待填写 |
| 打印头加热 | 不接入真实加热负载，或确保 VH 断开/关闭 | 待填写 |
| 测量工具 | 示波器或逻辑分析仪，所有探头共地 | 待填写 |

## 3. API 路径

| 阶段 | API | 期望 |
|---|---|---|
| 安全关闭 | `POST /api/v1/safe-off` | `G_VH=LOW`、`STB1~STB6=LOW`、电机输入低或安全态。 |
| Shift/Latch | `POST /api/v1/factory/head-shift-test`，请求体 48 字节 raw | 只看到 `DI/CLK/LAT` 波形，`G_VH` 保持 LOW，`STB` 不输出加热选通。 |
| STB 空载 | `POST /api/v1/factory/head-stb-test?group=0&pulse_us=5` | 指定 STB 有短脉冲，`G_VH` 保持 LOW。 |

## 5. 停止条件

- 任意时刻发现 `G_VH` 未保持关闭，立即停止后续测试。
- 任意 STB 在非测试组异常跳变，立即停止后续测试。
- 任意 GPIO 与文档映射不一致，先修正文档和 BSP 常量，不继续真实加热。
