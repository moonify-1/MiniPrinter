# MiniPrinter 引脚映射

更新时间：2026-04-26 21:34:32

## 1. 说明

- 本表基于 `项目背景/SCH_Schematic1_2026-04-24.pdf`、`项目背景/ESP32GPIO映射图.md`、`docs/硬件匹配分析.md` 和 `docs/硬件补充信息整理.md`。
- 用户已用万用表完成网络到 GPIO 连通性复核。
- 这些 GPIO 已可以作为固件常量落地，但真实输出行为仍必须通过空载安全态、示波器/逻辑分析仪波形和分阶段硬件测试验证。
- 真实热敏头、电机、VH 输出相关功能宏默认仍保持关闭。

## 2. 网络到 GPIO 映射

| 网络名 | GPIO | 方向 | 所属模块 | 有效电平 / 风险说明 |
|---|---:|---|---|---|
| `PAPER_N` | GPIO5 | 输入 | 缺纸检测 | LM393 输出；规格书倾向低电平表示有纸，但用户要求第一阶段先忽略缺纸检测。 |
| `G_NFAULT` | GPIO6 | 输入 | DRV8833 | 低有效故障输入；读到低电平表示电机驱动可能过流/过温/欠压。 |
| `G_NSLEEP` | GPIO7 | 输出 | DRV8833 | 低电平休眠，默认必须保持低以避免电机误动作。 |
| `G_KEY` | GPIO15 | 输入 | 按键 | 按键有效电平仍需用户确认；初期只做状态读取，不直接触发危险动作。 |
| `BAT_STAT` | GPIO16 | 输入 | IP2326 | 充电状态语义需继续按 IP2326 数据手册确认；未确认前固件返回保守状态。 |
| `EQD` | GPIO17 | ADC 输入 | 电池电压 | 2S+ 经过 220k/100k 分压，换算约为 ADC 电压的 3.2 倍；后续需要 ADC 校准。 |
| `G_VH` | GPIO8 | 输出 | 打印头电源 | 用户已确认高通低断；低电平关闭 VH，高电平打开 VH。默认必须低。 |
| `LAT` | GPIO3 | 输出 | 打印头 | 打印头锁存信号；空载验证前不得接入真实加热打印。 |
| `DI` | GPIO9 | 输出 | 打印头 | 打印头串行数据输入。 |
| `CLK` | GPIO10 | 输出 | 打印头 | 打印头串行时钟。 |
| `AIN1` | GPIO11 | 输出 | DRV8833 | A 相输入 1；默认低电平。 |
| `AIN2` | GPIO12 | 输出 | DRV8833 | A 相输入 2；默认低电平。 |
| `BIN1` | GPIO13 | 输出 | DRV8833 | B 相输入 1；默认低电平。 |
| `BIN2` | GPIO14 | 输出 | DRV8833 | B 相输入 2；默认低电平。 |
| `TM1` | GPIO2 | ADC 输入 | 打印头温度 | NTC 采样网络；等待用户提供 ADC 原始值后再做真实温度校准。 |
| `STB1` | GPIO40 | 输出 | 打印头 | 分组选通 1；高电平有效，默认低。 |
| `STB2` | GPIO39 | 输出 | 打印头 | 分组选通 2；高电平有效，默认低。 |
| `STB3` | GPIO38 | 输出 | 打印头 | 分组选通 3；GPIO38/RGB LED 风险已由用户确认不阻塞。 |
| `STB4` | GPIO37 | 输出 | 打印头 | 分组选通 4；高电平有效，默认低。 |
| `STB5` | GPIO36 | 输出 | 打印头 | 分组选通 5；高电平有效，默认低。 |
| `STB6` | GPIO35 | 输出 | 打印头 | 分组选通 6；高电平有效，默认低。 |

## 3. 后续验证要求

- 上电默认安全态：`G_VH=LOW`、`STB1~STB6=LOW`、`G_NSLEEP=LOW`、`AIN/BIN=LOW`。
- `G_VH` 虽已确认高通低断，但仍必须验证 `safe-off` 后保持低电平关闭。
- `STB1~STB6` 在未完成空载波形验证前，不得接入真实加热打印。
- `EQD` 换算先按 220k/100k 分压约 3.2 倍处理，最终电池电压仍需 ADC 校准。
